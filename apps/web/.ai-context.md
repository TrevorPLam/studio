# Web App - Studio

This is the Next.js application for managing AI agents, repositories, and sessions.

## Framework
- **Next.js** with App Router
- **React Server Components** by default
- **TypeScript** for type safety
- **shadcn/ui** for UI components

## Structure
- `app/` - Next.js App Router pages and API routes
- `components/` - React components (UI components in ui/ subfolder)
- `features/` - Feature modules (agents, repositories, sessions, github)
- `lib/` - Shared utilities, AI models, observability

## Key Features
- Agent management
- Repository management
- Session tracking
- GitHub integration
- Observability (metrics, tracing)

## Development
- Run: `npm run dev` or `pnpm dev`
- Build: `npm run build`
- Uses Server Components by default
