# Firebase Studio

A Next.js application for AI-powered agent sessions and GitHub repository management.

## Quick Start

1. See [SETUP.md](./SETUP.md) for detailed setup instructions
2. Install dependencies: `npm install`
3. Configure environment variables (see SETUP.md)
4. Run the development server: `npm run dev`

## Features

- AI agent sessions with streaming support
- GitHub repository integration
- Multi-model AI support
- Response caching and structured logging
- Authentication via NextAuth with GitHub OAuth
